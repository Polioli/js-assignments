Todo Lists Builder

Задание:

1) Есть кнопка добавления нового списка. Требуется реализовать класс ToDoBuilder, который будет создавать новый список ToDoList при инициализации и при нажатии на кнопку добавления. Создавать списки можно бесконечно. Как видно, для такой реализации потребуется разметку хранить/создавать внутри js кода

2) Для списка сделать кнопку, при нажатии на которую список будет очищаться.

Some notes how to use a todo-list

Input text and press Enter or click ok-button

To update item click on it, modify text and press Enter to confirm changes or Esc to cancel it
